## Usage

```tsx
import { GeneratedCommand } from "@/components/commandly/generated-command";
```

### Basic Usage

```tsx
const tool: Tool = {
  name: "curl",
  displayName: "curl",
  commands: [{ key: "curl", name: "curl", isDefault: true }],
  parameters: [
    {
      key: "url",
      name: "url",
      parameterType: "Option",
      dataType: "String",
      longFlag: "-X",
      isRequired: true,
      isRepeatable: false,
      isGlobal: false,
      commandKey: "curl",
    },
  ],
}

const [values, setValues] = useState({})

<GeneratedCommand
  tool={tool}
  selectedCommand={tool.commands[0]}
  parameterValues={values}
  onSaveCommand={(cmd) => console.log(cmd)}
/>
```

### With Save Handler

```tsx
<GeneratedCommand
  tool={tool}
  parameterValues={values}
  onSaveCommand={(cmd) => {
    navigator.clipboard.writeText(cmd);
    toast.success("Command saved!");
  }}
/>
```

## API Reference

### GeneratedCommand

| Prop              | Type                             | Default            | Description                                                  |
| ----------------- | -------------------------------- | ------------------ | ------------------------------------------------------------ |
| `tool`            | `Tool`                           | —                  | The tool definition including name, commands, and parameters |
| `selectedCommand` | `Command`                        | `tool.commands[0]` | The currently selected command to generate the string for    |
| `parameterValues` | `Record<string, ParameterValue>` | —                  | Map of parameter key to current value                        |
| `onSaveCommand`   | `(command: string) => void`      | —                  | Optional callback fired when the save button is clicked      |

## Notes

- The generated command updates reactively whenever `parameterValues` changes
- Flag parameters are only included when their value is `true`
- Option parameters are joined using `keyValueSeparator` (defaults to a space)
- Argument parameters are appended in order of their `position` field
- Global parameters are included regardless of the selected command
